¿QUE ES GIT?

para que sirve? Un sistema de control de versiones y en especial git tiene dos objetivos importantes: Guardar el historial de cambios sobre el código de un proyecto o varios y Permitir que un grupo de personas puedan contribuir en simultaneo o asincrónicamente en un mismo proyecto.

¿QUE ES CSS3?

Creo que la definición mas acertada para CSS es el de una hoja de de estilo con mucho estilo, aunque su historia se debe al boom del internet y la creciente popularidad del lenguaje de etiquetas htlm y la necesidad de un estándar para la definición de los estilos.

¿QUE ES HTML5?

El lenguaje de programación htlm (HyperTextMarkupLanguage) podría definirse mejor como un lenguaje de etiquetas con el que se puede construir documentos web o, propiamente dicho, paginas web de forma que los diferentes navegadores puedan entender nuestro contenido y lo puedan mostrar a los usuarios… y aunque no es el único que se puede usar, la inmensa mayoría de páginas están hechas en él.
